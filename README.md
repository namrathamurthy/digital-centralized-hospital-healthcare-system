# SmartCare Healthcare Management Platform

SmartCare is a comprehensive, fully integrated Next.js hospital suite designed to streamline medical operations, enhance patient care, and coordinate seamlessly across various clinical departments.

## Features

### Professional UI/UX Overhaul
- **Light Premium Theme:** A modern, calming, and professional interface tailored specifically for healthcare SaaS environments.
- **Glassmorphism & Responsive Design:** Elegant spacing, frosted-glass panels, smooth shadows, and soft gradients ensure a delightful user experience across all devices.

### Advanced Medical Billing & Payments
- **Dynamic Pricing Engine:** An automated billing system capable of generating itemized invoices instantly.
- **Payment Method Integration:** Support for various mock settlement workflows including:
  - Cash/UPI Gateway
  - Card Terminal Processing
  - Insurance Direct Billing
  - Bank Transfer

### Comprehensive Clinical Operations
- **Laboratory Hub & Tests:** Extensive tracking for over 75+ categorized lab scans and tests (Hematology, Biochemistry, Immunology, Radiology, etc.), fully integrated into the Doctor's prescription workflow.
- **Real-Time Queue Telemetry:** WebSocket-powered token synchronization across 6 operational dashboards (Patient, Doctor, Receptionist, Billing, Lab, Pharmacy).
- **Gemini AI Triage:** Intelligent symptom-checking algorithms to assess severity and route patients efficiently.

## Getting Started

First, run the development server or start the production build:

```bash
npm run dev
# or for production
npm run build
node server.js
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
